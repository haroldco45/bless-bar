# Bless Bar — PWA Granizados 🧊

Aplicación web progresiva (PWA) instalable para **Bless Bar**, Caucasia.

## Archivos
- `index.html` — App principal (imagen del poster embebida al 100%)
- `manifest.json` — Manifiesto PWA
- `sw.js` — Service Worker (modo offline)
- `icons/` — Iconos 192px y 512px

## Deploy (Netlify)
1. Subir todos los archivos a un repo GitHub
2. Conectar en netlify.app → New site from Git
3. Branch: main, Build: sin comando, Publish: /
